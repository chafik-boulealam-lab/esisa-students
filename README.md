## Quick checklist

- Form your group (3–5 students).
- Create a repo under `chafik-boulealam-lab` and invite Prof. Chafik Boulealam.
- Implement the core logic in C and integrate at least one AI feature.
- Use Supabase or MongoDB as the cloud database (do not commit credentials).
- Deploy the application and include the demo link and recorded demo in the README.
- Post the demo on LinkedIn and add the LinkedIn post URL in the README.
